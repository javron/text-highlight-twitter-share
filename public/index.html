Text Highlight & Twitter Share API

Getting Started

Include the following CSS and JavaScript files in your HTML page:

<link rel="stylesheet" href="https://text-highlight-twitter-share.vercel.app/quoteSharer.css" />
  <script src="https://text-highlight-twitter-share.vercel.app/quoteSharer.js"></script>

Add the "highlightable" class to the elements containing the text you want users to be able to highlight and share:

<p class="highlightable">Your text here...</p>
